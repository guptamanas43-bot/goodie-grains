# Pre-Deployment Checklist

Use this checklist to ensure your website is ready for deployment.

## Content Customization

### Brand Information
- [ ] Replace "Your Food Brand" with your actual brand name in all HTML files
  - [ ] index.html
  - [ ] about.html
  - [ ] products.html
  - [ ] bulk-orders.html

### Contact Information
- [ ] Update email address in footer (all pages)
- [ ] Update phone number in footer (all pages)
- [ ] Update contact details in bulk-orders.html

### Product Information
- [ ] Update product names in index.html (3 featured products)
- [ ] Update product descriptions in index.html
- [ ] Update product names in products.html (6 products)
- [ ] Update product descriptions in products.html
- [ ] Add actual Amazon product links (replace placeholder URLs)

### About Us Page
- [ ] Write your brand story
- [ ] Update founding year
- [ ] Customize values and mission
- [ ] Add any specific certifications or achievements

### Images
- [ ] Add product1.jpg to images/ folder
- [ ] Add product2.jpg to images/ folder
- [ ] Add product3.jpg to images/ folder
- [ ] Add product4.jpg to images/ folder
- [ ] Add product5.jpg to images/ folder
- [ ] Add product6.jpg to images/ folder
- [ ] Verify all images are optimized (under 500KB each)

## Optional Customizations

### Design
- [ ] Change color scheme in styles.css (if desired)
- [ ] Add your logo image
- [ ] Customize fonts (if desired)

### Features
- [ ] Set up Formspree for contact form (see README.md)
- [ ] Add Google Analytics tracking code (optional)
- [ ] Add social media links to footer (optional)

## GitHub Pages Deployment

- [ ] Create GitHub account (if you don't have one)
- [ ] Install Git on your computer
- [ ] Create new repository on GitHub
- [ ] Initialize git in project folder
- [ ] Commit all files
- [ ] Push to GitHub
- [ ] Enable GitHub Pages in repository settings
- [ ] Wait for deployment (1-5 minutes)
- [ ] Test your live website

## Post-Deployment

- [ ] Test all navigation links
- [ ] Test all Amazon product links
- [ ] Check mobile responsiveness
- [ ] Test on different browsers (Chrome, Firefox, Safari)
- [ ] Verify images load correctly
- [ ] Test contact form (if integrated)
- [ ] Share your website URL!
